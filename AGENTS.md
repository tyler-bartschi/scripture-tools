# Instructions for all AI agents working on this project

## Attributions

For each prompt that results in changes to a file (any file, not just code files) please append a statement to the
AGENT_ATTRIBUTIONS.md file located at the root of the project. Each statement in the AGENT_ATTRIBUTIONS.md file
should follow the following format:

*Date*
- Summary of the prompt (no more than 150 characters)
- Concise summary of changes made
- Any additional notes deemed appropriate for recording (this should be minimal)

You are required to append a statement like the one above for each prompt that results in file or directory changes.
You are not required to append a statement if the prompt resulted in no file or directory changes (such as simply
answering a user's question). Additionatlly, please ensure each attribution is separated by a blank line, for ease of reading.
